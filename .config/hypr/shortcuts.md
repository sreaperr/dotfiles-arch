# Atajos de teclado

## Aplicaciones
| Atajo | Acción |
|---|---|
| `Super + Enter` | Terminal (kitty) |
| `Super + B` | Navegador (Brave) |
| `Super + Space` | Launcher apps |
| `Super + Y` | Gestor de archivos (yazi) |
| `Super + K` | Calendario (calcurse) |
| `Super + H` | Portapapeles |

## Ventanas
| Atajo | Acción |
|---|---|
| `Super + W` | Cerrar ventana |
| `Super + F` | Pantalla completa |
| `Super + Shift + F` | Flotar / destilar |
| `Super + Ctrl + F` | Centrar y redimensionar |
| `Super + P` | Fijar encima (pin) |
| `Super + M` | Intercambiar con master |
| `Super + ←↑↓→` | Mover foco |
| `Super + Shift + ←↑↓→` | Mover ventana |
| `Super + Ctrl + ←↑↓→` | Redimensionar |

## Sistema
| Atajo | Acción |
|---|---|
| `Super + L` | Bloquear pantalla |
| `Super + I` | Apagar / reiniciar |
| `Super + O` | Voz (Reep) |
| `Super + R` | Ciclar tema |
| `F3` | Redistribuir ventanas |
| `F4` | Paneles flotantes |
| `F11 / F12` | Volumen − / + |
| `F10` | Silenciar |
| `Print` | Captura de pantalla |

## Workspaces
| Atajo | Acción |
|---|---|
| `Super + 1-9` | Ir al workspace |
| `Super + Shift + 1-9` | Mover ventana |
| `Super + Scroll` | Navegar workspaces |
